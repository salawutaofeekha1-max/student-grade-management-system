# GPA Calculator (COS 102 Project)

A simple command-line Python program that helps a student calculate their semester GPA.

## Features
- Add courses with title, unit load, and grade
- View all courses entered so far
- Automatically calculate semester GPA using the 5-point grading scale
- Save course records to a CSV file (`gpa_records.csv`) so data is not lost when the program closes
- Automatically loads saved records the next time the program runs

## Grading Scale Used
| Grade | Points |
|-------|--------|
| A     | 5      |
| B     | 4      |
| C     | 3      |
| D     | 2      |
| E     | 1      |
| F     | 0      |
